smrate.exe предназначена для управления режимами работы пробоотборного устройства.

smrate.exe использует промышленный контроллер ОВЕН МУ110-224.16р.

Связь ПК с МУ110-224.16р осуществляется по RS-232 (СОМ порт).

smrate.exe – 32-битное консольное приложение Windows.

smrate.exe запускается из консоли. Для работы со smrate.exe необходимо:
    - запустить консоль Windows:
        - нажать значок поиска на Панели задач или кнопку Пуск
        - В строке поиска напечатать cmd
        - В результатах поиска нажать на классическом приложении Командная строка
    - установить рабочий каталог приложения:
        - в командной строке ввести:
            chdir /d [путь к каталогу, в котором находится smrate.exe]
    - установить СОМ порт:
	    - выполнить в командной строке:
	        SET COMPORT=[имя компорта, к которому подключено устройство ОВЕН МУ110-224.16р].
	        Чтобы получить список доступных СОМ портов, необходимо выполнить команду smrate ports.

Аргументы командной строки smrate.exe:
	- (без аргументов) : однократно выполниить сценарий smrate.txt
	- loop : многократно выполниить сценарий smrate.txt
	- close : закрыть все клапаны
	- ports - распечатать список доступных СОМ портов
	- valve [N] : установить состояния клапанов, где N - 10 бит в двоичном формате
	- run [loop - опционально, повторять многократно] [N1] [P1] [N2] [P2]...[Nn] [Pn] :
		для i от 1 до n:
			- установить состояния клапанов Ni, где Ni - 10 бит в двоичном формате
			- выдержать паузу Pi, где Pi - длительность паузы, знаковая последовательность десятичных чисел,
				каждое из которых содержит необязательную дробь и суффикс единицы времени,
				например "300ms", "-1.5h" или "2h45m".
				Допустимые единицы времени: "ns", "us" (или "µs"), "ms", "s", "m", "h".

Переменные окружения, используемые smrate.exe:
	- COMPORT : имя СОМ порта связи
	- COMPORT_LOG=true : показывать в консоли посылки СОМ порта

Текстовый файл smrate.txt представляет собой сценарий работы smrate.exe при запуске без аргументов, либо с аргументом loop.

Каждая строка текстового файла smrate.txt содержит инструкцию однократной установки состояния клапанов пробоотборного устройства и выдержки требуемой паузы:
    - первое слово в строке - код сосотяния клапанов, десятибитное число в двоичном формате
    - вотрое слово - длительность паузы, аналогично run
    - далее - опционально, текст кометария, выводимый программой при выполнении данной инструкции
